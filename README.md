# glados自动签到，实现无限白嫖。

环境变量：`GLADOS_COOKIE`(必要) 和 `SERVERCHAN`(非必要)

 `GLADOS_COOKIE`多个账号需使用 '&' 隔开
  示例：cookie&cookie&cookie



# 食用方法 （github_actions）

 1.点击右上角 **fork** 按钮 
 
 2.在自己仓库中打开此项目
 
 3.配置环境变量
 
 4.点亮右上角的星星 **star** 激活 actions
 
 5.然后点击 Actions 标签查看运行的详细状况
